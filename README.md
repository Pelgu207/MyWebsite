# MyWebsite

## Live Website

Visit the site at: **https://pelgu207.github.io/MyWebsite/**

The website is hosted on GitHub Pages and is automatically deployed whenever changes are pushed to the `main` branch. No need to manually run or host anything.

## Setup

To enable GitHub Pages for this repository:

1. Go to **Settings** → **Pages** in your GitHub repository.
2. Under **Build and deployment**, set the **Source** to **GitHub Actions**.
3. Push changes to the `main` branch — the site will deploy automatically.
